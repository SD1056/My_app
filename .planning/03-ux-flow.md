# UX 플로우 & 화면 설계

## 화면 목록

| # | 화면 | 설명 |
|---|------|------|
| S1 | Splash | 앱 로딩 |
| S2 | Onboarding | 첫 실행 시 3장 슬라이드 |
| S3 | 홈 | 오늘 기록 요약 + 최근 기록 목록 |
| S4 | 기록 입력 | + 버튼 → AI 제안 or 직접 입력 |
| S5 | 기록 히스토리 | 날짜별 전체 기록 조회 |
| S6 | 통계 | 주간·월간 차트 |
| S7 | 설정 | 카테고리 관리, 알림 설정 |

---

## 전체 앱 플로우

```
앱 실행
  │
  ├─ 첫 실행? ──YES──► S2 온보딩 (스킵 가능)
  │                         │
  │                         ▼
  └─ 재실행 ──────────► S3 홈 화면
                            │
          ┌─────────────────┼─────────────────┐
          ▼                 ▼                 ▼
     [📝 기록]          [📊 통계]         [⚙️ 설정]
       S4/S5               S6               S7
```

## 화면별 상세 플로우

### S2 — 온보딩 (첫 실행)

```
┌─────────────────────┐  →  ┌─────────────────────┐  →  ┌─────────────────────┐
│   [Skip]            │     │   [Skip]            │     │                     │
│                     │     │                     │     │                     │
│      📱             │     │      🤖             │     │      ✅             │
│                     │     │                     │     │                     │
│  매일 반복되는       │     │  AI가 패턴을         │     │  카테고리를 선택하고  │
│  기록을 자동화해요   │     │  학습해서 대신        │     │  지금 바로 시작해요  │
│                     │     │  제안해드려요        │     │                     │
│  ● ○ ○             │     │  ○ ● ○             │     │  ○ ○ ●             │
│                     │     │                     │     │                     │
│      [다음]         │     │      [다음]         │     │   [시작하기]        │
└─────────────────────┘     └─────────────────────┘     └─────────────────────┘
슬라이드 1                    슬라이드 2                    슬라이드 3
```

- 우상단 [Skip] 탭 → 즉시 홈으로 이동
- 슬라이드 3 [시작하기] → 홈으로 이동

### S3 — 홈 화면

```
┌─────────────────────┐
│  Myapp       🔔     │  ← 알림 아이콘
├─────────────────────┤
│  오늘 2026.05.18    │
│  ┌─────────────┐   │
│  │ 기록 3건    │   │  ← 오늘 요약 카드
│  │ 총 13,500원 │   │
│  └─────────────┘   │
├─────────────────────┤
│  최근 기록          │
│  ─────────────────  │
│  ☕ 커피  4,500원   │  09:12
│  🍱 점심 8,000원   │  12:30
│  💪 운동  완료      │  19:00
│                     │
│  [더보기 →]        │
├─────────────────────┤
│  🏠   📝   📊   ⚙️ │
└─────────────────────┘
```

- 요약 카드 탭 → S6 통계
- 기록 항목 탭 → 해당 기록 상세/수정
- [더보기] → S5 히스토리
- 📝 탭 or FAB(+) → S4 기록 입력

### S4 — 기록 입력 플로우

```
[+] 버튼 탭
      │
      ▼
 AI 제안 있음?
      │
  YES ▼                    NO ▼
┌──────────────────┐   ┌──────────────────┐
│  🤖 AI 제안      │   │  카테고리 선택   │
│  ─────────────   │   │  ─────────────   │
│  ☕ 커피          │   │  💰 지출         │
│  4,500원         │   │  💪 운동         │
│  오전 9시 추정   │   │  🍱 식단         │
│                  │   │  + 직접 추가     │
│  [✓ 수락]        │   └──────────────────┘
│  [✎ 수정]        │            │
│  [✕ 건너뜀]      │            ▼
└──────────────────┘   ┌──────────────────┐
      │                │  값 입력         │
  수락 ▼               │  (텍스트/숫자)   │
 즉시 저장             │  [저장]          │
      │                └──────────────────┘
  수정 ▼
┌──────────────────┐
│  인라인 수정     │  ← 금액/내용 직접 편집
│  [저장]          │
└──────────────────┘
      │
      ▼
  저장 완료 → 홈으로 복귀 + 스낵바 "기록됐어요 ✓"
```

**AI 제안 카드 규칙:**
- 과거 30일 패턴에서 현재 시간대와 일치하는 항목이 있을 때만 표시
- 제안이 2개 이상이면 스와이프로 전환
- 건너뜀 3회 연속 → 해당 패턴 가중치 감소

### S5 — 기록 히스토리

```
┌─────────────────────┐
│  ← 기록 전체       │
├─────────────────────┤
│  [월] [주] [일]     │  ← 뷰 전환 탭
├─────────────────────┤
│  2026년 5월         │
│                     │
│  18일 (오늘)        │
│  ├ ☕ 커피 4,500    │
│  ├ 🍱 점심 8,000   │
│  └ 💪 운동 완료     │
│                     │
│  17일 (어제)        │
│  ├ ☕ 커피 4,500    │
│  └ 🍱 저녁 12,000  │
└─────────────────────┘
```

- 항목 길게 누르기 → 수정 / 삭제 옵션

## 핵심 인터랙션 원칙

| 원칙 | 적용 |
|------|------|
| 원탭 수락 | AI 제안 수락은 버튼 1번으로 완료 |
| 즉각 피드백 | 저장 후 스낵바로 확인 메시지 표시 |
| 실수 복구 | 저장 직후 3초간 [취소] 옵션 제공 |
| 빈 상태 안내 | 데이터 없을 때 "첫 기록을 남겨보세요" 안내 |
| 로딩 최소화 | AI 제안은 백그라운드 프리페치, 로딩 스피너 없음 |
