# 비전 & 목표

## 한 줄 비전

> "기록하는 습관 없이도, AI가 대신 패턴을 기억한다."

## 문제 정의

사용자들은 지출, 건강, 일정 등 반복적인 데이터를 매일 손으로 입력한다.
이 과정은 귀찮고 빠뜨리기 쉬우며, 결국 데이터가 쌓이지 않아 분석도 못 한다.

## 제품 비전

AI가 사용자의 데이터 입력 패턴을 학습하여,
다음 번 입력을 자동으로 제안하거나 완성해주는 모바일 앱.
사용자는 "확인"만 눌러도 기록이 완성된다.

## 목표 (OKR)

### Objective 1 — 핵심 자동화 경험 검증 (MVP)
- KR1: 첫 3일 사용 후 자동 제안 수락률 ≥ 60%
- KR2: 일일 입력 소요 시간 기존 대비 50% 단축
- KR3: 7일 연속 사용자 리텐션 ≥ 40%

### Objective 2 — 사용자 신뢰 확보
- KR1: AI 제안 정확도 (사용자 수정 없이 수락) ≥ 70% (30회 사용 이후)
- KR2: 앱스토어 평점 ≥ 4.2

### Objective 3 — 초기 성장
- KR1: 론칭 3개월 내 MAU 1,000명
- KR2: D30 리텐션 ≥ 25%

## 성공 기준

| 단계 | 기준 |
|------|------|
| MVP  | 자동 제안이 실제로 시간을 아껴준다는 걸 5명이 검증 |
| 베타 | DAU 100명이 매일 앱을 열고 제안을 수락 |
| 정식 | 유료 전환율 ≥ 5% |

## 범위 제외 (v1)

- 웹 버전
- 팀/공유 기능
- 서드파티 앱 자동 연동 (추후 v2)
