문제

여러분은 매일 같은 데이터를 반복 입력합니다.
- 지출: 같은 카페 3만원
- 운동: 같은 코스 5km
- 식단: 같은 메뉴

해결책

AI가 이 패턴을 학습해서 미리 제안합니다.

가치

3초 → 1초로 줄어듭니다.
연 24시간 절약할 수 있습니다.

---

실제 사용하면 이렇습니다:

1단계: 사용자가 + 버튼을 누릅니다.

2단계: AI가 지난 30일 기록을 분석합니다.
"아, 이 사람은 매일 5km씩 달렸네"

3단계: AI가 제안합니다.
"오늘도 운동 5km 할 거 같아?"

4단계: 사용자는 3가지 중 선택합니다.
- "응" → 바로 저장
- "3km로 해줘" → 수정해서 저장
- "건너뛸래" → 직접 입력

전체 시간? 2~3초입니다.

[SD1056 수정: "응"의 결과를 더 명확하게 표현하기 위해 "바로 저장" → "1초만에 저장" 으로 변경 검토 중]
